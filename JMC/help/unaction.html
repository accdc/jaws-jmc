#unaction

Syntax: #unaction {<action to be deleted>}

Removes a (pre)defined action. The pattern of the removing action must be used as parameter. You can also use wildcards to remove a few actions in one time.

Example:

#unaction {crashes into you}

Remove an action with pattern "crashes into you".

#unaction {*you*}

Removes all actions with "you" in the pattern.

See also: #action

Profile Options

This page of the "Options" dialog box allows the user to set up options detailing the currently loaded profile. There are three options here:

  • The name of the macro file: This is the file that JMC reads the profile in from when JMC boots up.
  • The name of the save file: When JMC shuts down, the profile currently in use is saved to this file.
  • Exit command: This is a JMC command that will be executed before the profile deloads (for example, closing a log file).

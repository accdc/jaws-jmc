#message

Syntax: #message {type} [ON/OFF]

This command toggles whether messages are displayed on screen while you define/delete aliases, variables, substitutions, antisubstitutions, or highlights. Valid values for "type" are: alias, action, substitute, antisubstitute, highlights, variable groups.

This command is useful when you don't want to spam yourself while you set up aliases, triggers, variables, actions etc.

Second parametr of this command is not necessary. Valid values is "ON" or "OFF". If you dont use it JMC will change current message state to the inversed state.

Example:

#message variable OFF

You will no longer see messages like this:

#Variable start now set to 1

See also: #info

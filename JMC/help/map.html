#map

Syntax: #map <direction>

This command will add a direction to the end of the path currently being stored. It's useful for recording the directions when following someone.

Example:

#action {You follow Lesath %0.} {#map {%%0}}

If Lesath leaves the room, the direction is added to the end of the current path.

See also: #mark, #pathdir, #unpath, #path, #savepath, #return

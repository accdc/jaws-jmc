#output

Syntax: #output [{color }] {<text>}

This command will display <text> on the additional window (“output” window) by the specified color. Color isn’t obligatory parameter. #output may be used in both forms:

  • #output This is just text displayed by the default color
  • #output {red} {this text will be displayed buy red color}

Colors specifications are same as in the #highlight command

Example:

#action{%0 decays in your hands} {#output {*** %0 decayed !!!}}

Now, you will see a message "sword decayed !!!" every time a sword decays in your hands.

See also: #highlight #showme

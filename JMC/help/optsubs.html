Substitution page

This page of the "Options" dialog allows you to set up character substitution. Character substitution basically allows you to convert a character coming from the MUD to another character. This is mostly used for MUDs in foreign languages, where certain keycodes have different meanings. You can turn substitution on/off and set up character mapping.

The list of substitutions shows all characters currently being mapped. Two edit controls show the source and resulting character of the current substitution pair. The "Add" button adds a character pair to the list. The "Remove" button removes the currently selected pair from the list.

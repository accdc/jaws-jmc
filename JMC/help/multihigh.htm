#multihighlight

Syntax: #multihighlight [on/off]

Toggles "multihighlight mode" ON/OFF. When "multihighlight mode" is OFF, JMC stops highlighting the incoming string after the first match  is found. Sometimes, however, you have to apply multiple highlights to the one string. When the "multihighlight mode" is ON, JMC checks all highlights on every incoming string.

See also: #highlight

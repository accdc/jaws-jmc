#drop

Syntax: #drop

This command may be used only in the triggers content. Use this command to prevent displaying triggered line.

Sample:

#action {%0 crashes into you} {#showme YOU BASHED!!!!;sta;drop}

Result of this is – every time line «Dragon crashes into you» arrives it will be triggered, «YOU BASHED» string will be displayed, but origimnal line will not displays.

See alse: #action #gag

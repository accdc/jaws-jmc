Load profile dialog

This dialog is displayed when a user chooses to load a profile. It contains a list of all profiles that JMC knows about. When you choose a profile to load, the old (current) profile's macros are saved to a file, executing JMC's exit command (if its defined - see Profile Options). After JMC has cleared all macros, highlights, actions and related components, the chosen profile is loaded.

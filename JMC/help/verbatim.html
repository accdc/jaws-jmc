#verbatim

Syntax: #verbatim [on/off]

Toggle verbatim mode on and off. When in verbatim mode, user input will not be parsed, and will be sent 'as is' to a MUD. It is helpful for writing messages, doing online creation, and the like.

Second parameter of this command is not necessary. Valid values is "ON" or "OFF". If you don’t use it JMC will change current verbatium state to the inverted state.

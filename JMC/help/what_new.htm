What is new in JMC 3.* .

 

Well. If you used JMC 2.15 long time and downloaded JMC 3.* first time, you need some info about futures, implemented.

First, interface changed a lot. JMC now able to split main window while backscroll, so you may keep looking for current situation in the MUD while backscrolling buffer. Also, added extended window (its dockable and may be attached to any part of JMC window or moved out of it). You able to display any text in this window by #output command.

JMC now able to save logs into HTML form. Look #log for more info.

ActiveScripting support added ! Now you able to make real complex analyze of incoming text and do very complex actions, to use databases etc.

Groups of commands implemented.

Statusbar of the JMC main window changed – its have new form and got 3 cells with user-defined text in it. Look at #status command.

A lot of command’s format changed (for compatibility, old formats still accepted, just added new parameters ):

#alias #action #highlight  #hotkey #ignore #log #message #presub #showme #speedwalk #substitute #togglesubs #variable #verbatim #write

Well. A lot of futures and changes. I am pretty sure all your old scripts will work OK on new JMC. Look at menu items and dialogs for more new futures. Good luck.

What is RMA?

Basically, it is simple ANSI player with one new code added: [ESC]p:[millisecs]m , where [ESC] is escape code, [millisecs] amount of time to pause

RMA gives you an ability to view MUD logs (battles, challenges etc) in full realism, with colors and pauses. Give it a try and you will surely like it.

This version of RMA is distributed within JMC MUD client by Sergey Ilin. It can be used to view RMA LOG files generated by JMC or RMUD server. To view such file, just type: rma.exe [filename]. You can run rma.exe with no parameters to read the help.

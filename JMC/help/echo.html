#echo

Syntax: #echo [on/off]

This command toggles echoing of action commands. While echo mode is ON, all triggered action will be displayed on your screen. If echo mode is OFF, then the commands will be "silently" sent to the mud without showing it on screen.

Second parametr of this command is not necessary. Valid values is "ON" or "OFF". If you don’t use it JMC will change current echo state to the inverted state.

Example:

#echo

Turns echo on/off.

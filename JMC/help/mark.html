#mark

Syntax: #mark {start|stop}

This command takes one argument which may be skipped. If the argument is "start" (or non-existant) then this command clears the current saved path and turns "path mode" ON. When path mode is on, JMC automatically keeps all typed direction commands in a list. When you turn this mode off, using the "stop" argument, JMC stops recording the path.

Example:

#mark

All saved directions cleared and path mode turn ON.

See also: #map, #pathdir, #unpath, #path, #savepath, #return

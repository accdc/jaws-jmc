#togglesubs [on/off]

Syntax: #togglesubs

Similar to #ignore, #togglesubs will toggle substitutions processing ON/OFF. The default state is ON. If switched off, no substitution of incoming text will be performed.

Second parameter of this command is not necessary. Valid values is "ON" or "OFF". If you don’t use it JMC will change current togglesub state to the inverted state.

 

See also: #substitute
